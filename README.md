## Come mi piace lavorare

Quando sviluppo cerco sempre di tenere insieme tre cose:

1. **Chiarezza**  
   Il codice deve essere comprensibile anche dopo settimane o mesi.

2. **Concretezza**  
   Una feature deve funzionare bene per l’utente finale, non solo “passare il test”.

3. **Crescita continua**  
   Mi piace imparare mentre lavoro. React, Next.js e diversi aspetti full-stack li ho approfonditi direttamente su progetti reali, adattandomi alle necessità del prodotto.
